HTTP Request

GET {{url}}/api/MessageThread

Description

Returns messages by user.

  • Set user1 to a user's username to get a list of all messages sent to and from that user. Messages will be sorted into threads (messages between two users) and then sorted by time sent.
  • Set user2 to another user's username to get the thread of messages between user1 and user2.

Parameters

ParameterRequiredTypeDescription
user1Yesstringusername (must exist in database)
user2Nostringusername (must exist in database)

Examples

Template

curl -i "http://localhost:5000/api/MessageThread?user1=[username1]&user2=[username2]

Input

curl -i "http://localhost:5000/api/MessageThread?user1=ecroby"

Output

[
{ "body": "hi", "from": "ecroby", "message_id": 1, "thread": "ecroby - user2", "time": "Fri, 26 Aug 2016 21:51:02 GMT", "to": "user2" }
, { "body": "hello there!", "from": "user2", "message_id": 2, "thread": "ecroby - user2", "time": "Fri, 26 Aug 2016 21:51:18 GMT", "to": "ecroby" }
, { "body": "nice to meet you", "from": "ecroby", "message_id": 5, "thread": "ecroby - user3", "time": "Sun, 28 Aug 2016 14:53:56 GMT", "to": "user3" }
]

Input

curl -i "http://localhost:5000/api/MessageThread?user1=ecroby&user2=user2"

Output

[
{ "body": "hi", "from": "ecroby", "message_id": 1, "thread": "ecroby - user2", "time": "Fri, 26 Aug 2016 21:51:02 GMT", "to": "user2" }
, { "body": "hello there!", "from": "user2", "message_id": 2, "thread": "ecroby - user2", "time": "Fri, 26 Aug 2016 21:51:18 GMT", "to": "ecroby" }
]







HTTP Request

POST {{url}}/api/MessageThread

Description

Posts message and returns thread between two users including the posted message.

  • Set from, to and body to post message.

Parameters

ParameterRequiredTypeDescription
fromYesstringusername (must exist in database)
toYesstringusername (must exist in database)
bodyYesstringmessage body

Examples

Template

curl -i -H "Content-Type: application/json" -X POST -d '{"from":"[username]", "to": "[username]", "body": "[body text]"}' http://localhost:5000/api/MessageThread

Input

curl -i -H "Content-Type: application/json" -X POST -d '{"from":"ecroby", "to": "user2", "body": "Bruh, u watched Point Break yet?"}' http://localhost:5000/api/MessageThread

Output

[
{ "body": "hi", "from": "ecroby", "message_id": 1, "time": "Fri, 26 Aug 2016 21:51:02 GMT", "to": "user2" }
, { "body": "hello there!", "from": "user2", "message_id": 2, "time": "Fri, 26 Aug 2016 21:51:18 GMT", "to": "ecroby" }
, { "body": "Bruh, u watched Point Break yet?", "from": "ecroby", "message_id": 13, "time": "Sun, 28 Aug 2016 21:49:11 GMT", "to": "user2" }
]
